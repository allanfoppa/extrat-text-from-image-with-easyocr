# Extrat text from image with easyocr

## Create VENV
```
python3 -m venv path/to/your/project
```

### Activate
```
source bin/activate
```

### Deactivate
```
deactivate
```

## Update requirements.txt
```
pip3 freeze > requirements.txt
```
